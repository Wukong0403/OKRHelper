# OKR助手 📊

一款简洁高效的季度OKR规划与任务管理Android应用，帮助你更好地制定和追踪目标。

## ✨ 功能特性

### 🎯 基于关键结果的四级任务体系
- **季度OKR管理**：设定季度目标(Objectives)
- **关键结果拆分**：为每个OKR添加多个关键结果(Key Results)
- **月度任务规划**：针对每个关键结果制定月度执行计划
- **周任务细化**：将月度任务进一步细化到每周具体行动

### 📱 直观的用户界面
- **双Tab设计**：OKR列表 + 日历视图，一键切换
- **可展开列表**：四级层级展开，点击即可展开/收起
- **任务状态追踪**：圆形图标标记完成状态，已完成任务有明确标注
- **Material 3 设计**：现代化的UI设计，美观流畅
- **四层卡片配色**：不同颜色区分不同层级（蓝-青-橙-渐变）

### 📅 日历视图
- **月度视图**：查看当月所有待办任务
- **周度视图**：聚焦本周任务，更加专注
- **一键切换**：月/周视图快速切换
- **直观展示**：清晰显示任务数量和完成状态
- **快速操作**：点击任务卡片即可标记完成

### 💾 数据持久化
- **本地存储**：使用Room数据库，数据安全可靠
- **自动保存**：所有操作即时保存，无需担心数据丢失
- **级联删除**：删除上层任务时自动清理关联的子任务

## 🖼️ 界面展示

### OKR列表页
应用采用四层卡片折叠设计：
- 🟦 蓝色卡片：季度OKR（最外层）
- 🟩 青色卡片：关键结果
- 🟨 橙色卡片：月度任务
- 🟪 浅色卡片：周任务（最内层）

点击卡片即可展开查看详细信息和下级任务。每个层级都可以独立标记完成状态。

### 日历页
- 顶部显示当前月份/周数和任务统计
- 任务卡片采用圆形图标，已完成任务有"已完成"标签
- 支持月视图和周视图一键切换
- 空状态提示清晰友好

## 🚀 使用方法

### OKR列表页

#### 创建OKR
1. 在OKR Tab页，点击右下角的 ➕ 按钮
2. 填写年份、季度和目标
3. 点击确定保存

#### 添加关键结果
1. 点击OKR卡片展开
2. 在"关键结果"区域点击"添加"按钮
3. 填写关键结果标题和描述（可选）

#### 添加月度任务
1. 展开OKR卡片，再展开关键结果卡片
2. 在"月度任务"区域点击"添加"按钮
3. 填写年份、月份、任务标题和描述

#### 添加周任务
1. 逐级展开至月度任务
2. 在"周任务"区域点击"添加"按钮
3. 填写年份、周数、任务标题和描述

### 日历页

#### 查看任务
1. 切换到"日历"Tab
2. 默认显示当月任务
3. 点击右上角图标切换周视图/月视图

#### 标记完成
- OKR列表：点击任务卡片右侧的圆形图标标记完成
- 日历页：直接点击任务卡片即可标记完成/未完成

### 切换视图
点击底部导航栏在"OKR"和"日历"之间切换。

## 🛠️ 技术栈

- **语言**：Kotlin
- **UI框架**：Jetpack Compose
- **架构模式**：MVVM
- **数据库**：Room
- **导航**：Navigation Compose (已移除，采用单页面设计)
- **异步处理**：Kotlin Coroutines + Flow
